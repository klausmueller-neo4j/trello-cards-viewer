## Overview

Trello Cards Viewer is a Vue 3 application built with Vite and Vuetify. It allows you to select a Trello board member and view their card creation and copy actions. Cards are displayed in a searchable and paginated list with detailed information loaded via batch API calls.

## Features

- **Member Filtering:** Select a member from your Trello board.
- **Action Pagination:** Fetch actions in manageable batches using pagination.
- **Batch Card Details:** Retrieve detailed information for multiple cards concurrently.
- **Search & Filter:** Filter cards by title and description.
